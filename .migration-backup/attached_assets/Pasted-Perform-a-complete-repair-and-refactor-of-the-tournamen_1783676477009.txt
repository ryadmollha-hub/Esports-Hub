========================================
MATCH CREATION
========================================

Creating a Match should ONLY:

• create Match #1 / Match #2 etc.
• save scheduled match time
• save match name
• save match number

It must NOT

• release room
• close registration
• mark match live
• show room id
• show password
• hide room
• complete match

========================================

========================================

========================================
REGISTRATION
========================================

Registration remains OPEN until Room Release.

Only after Room Release:

• Registration Closed
• Join button disabled
• Existing registered players remain.
