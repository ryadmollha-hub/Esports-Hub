==================================================
ISSUE 1: COMMUNITY MATCHES NOT APPEARING
========================================

Problem:

User-created public matches are being successfully created and saved.

However, they are not appearing inside the Community Matches section on the Tournament page.

Required Fix:

Display all public community matches automatically inside the Community Matches section.

Requirements:

* Public community matches must appear immediately after creation.
* Match cards must update automatically.
* Community Matches section must load real database data.
* Do not require manual approval.
* Do not hide valid public matches.

Display:

* Match Number
* Match Name
* Match Mode
* Host Name
* Entry Fee
* Prize Pool
* Player Count
* Start Time
* Join Button

Private Matches:

* Also display them.
* Show a Private Badge.
* Do not reveal passwords.
* Allow authorized users to join using password.

MATCH COMPLETION & AUTO HIDE SYSTEM
===================================

Problem:

After a match result is entered and marked as completed, the match still remains visible in active tournament lists and management sections.

Required Behavior:

When Admin clicks:

"Save Results & Mark Complete"

Automatically:

* Match status becomes COMPLETED
* Match is removed from all active tournament pages
* Match is removed from active player join pages
* Match is removed from upcoming matches
* Match is removed from active tournament listings
* Match can no longer be joined

Completed matches should only appear in:

* Results Page
* Match History
* Archived Matches
* Admin Records

Do not permanently delete match data.

Keep all records for future reference.

DELETE SYSTEM (TWO LEVELS)
==========================

Level 1:

COMPLETE → ARCHIVE

Recommended default action.

Behavior:

* Match disappears from active sections.
* Match remains in Archive.
* Results remain available.
* History remains available.
* Rankings remain available.
* Room history remains available.

Level 2:

PERMANENT DELETE

Add:

🗑 Delete Match

button on every Match Management Card.

Before deletion show:

"Are you sure you want to permanently delete this match?"

Buttons:

* Cancel
* Confirm Delete

Permanent Delete removes:

* Match Information
* Rankings
* Room Details
* Room Credentials
* Player Registrations
* Match Results
* Match History
* Associated Records

Deleted matches cannot be restored.
