* Players Button
* Room ID Release System

* Room Close Time

6. MATCH SERIAL NUMBER SYSTEM

Implement a permanent auto-generated Match Number system.

This is required because admins manage many matches and need a simple way to identify matches when publishing results, reviewing matches, or managing rooms.

Requirements:

* Every match must receive a permanent unique Match Number.
* Match Numbers must be generated automatically.
* Match Numbers must never repeat.
* Match Numbers must never reset.
* Deleted matches must not reuse old numbers.
* Numbers must continue increasing forever.

Recommended Format:

Admin Tournament Matches:

T-0001
T-0002
T-0003

Community Matches:

C-0001
C-0002
C-0003

Display Match Number on:

* Tournament Cards
* Community Match Cards
* Tournament Details
* Community Match Details
* Admin Tournament Management
* User Match Management
* Results Page
* Match History
* Players Page

Search System:

Admin must be able to search matches by:

* Match Number
* Match Name

Result Publishing:

Always display:

Match Number
Match Name
Match Type

Example:

T-0125
Friday BR Cup
BR Tournament

This ensures admins never get confused when managing hundreds or thousands of matches.
